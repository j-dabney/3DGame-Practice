# 3D Game Practice
This is 3D game practice using C++ w/ Raylib.

## How to Build
```mkdir Build```
```cd Build/```
```cmake ..```
```make```
### For clangd support:
```cd ..```
```ln -s Build/compile_commands.json```

You have now built the project.

Any files added must be reflected in the directories respective CMakeLists.txt and the above process repeated.
